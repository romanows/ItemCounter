public class ItemDoubleAccumulator<K>
extends java.lang.Object

Accumulates values associated with items.

The behavior of this class differs conceptually from ItemCounter. In ItemCounter, any object that is not contained in the ItemCounter would be reported to have a count of zero. In the ItemDoubleAccumulator, an object that has never been added to the accumulator will cause get(Object) to return a null. An object that has been added with a value of zero will cause get(Object) to return zero.
