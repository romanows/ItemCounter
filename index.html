ItemCounter by romanows

Simple Java objects to count items and accumulate values associated with items

Dependencies

Java 1.5+, JUnit 4 to run the unit tests.

Install

Copy the ItemCounter.java and/or ItemDoubleAccumulator.java files into your project. The GitHub repository is itself an Eclipse project that may be imported into an Eclipse workspace.
